# 🌾 Green26 - Conectando Campo e Tecnologia

> Projeto desenvolvido por **Rafael das Chagas de Oliveira** para o prestigiado concurso **Agrinho 2026**.

---

## 🎯 Objetivo do Projeto
O **Green26** nasceu com a missão de estreitar os laços entre o consumidor urbano e o produtor rural. O site busca conscientizar a sociedade sobre a realidade do campo moderno em Rio Bonito do Iguaçu e no Paraná, demonstrando como a agricultura de precisão, a tecnologia verde e o cooperativismo familiar andam de mãos dadas com a preservação ambiental para alimentar o mundo de forma sustentável.

---

## 🚀 Recursos Técnicos e Acessibilidade
Para garantir uma experiência inclusiva, moderna e fluida para todas as pessoas usuárias, o site conta com os seguintes recursos:
* 🌙 **Modo Escuro (Dark Mode):** Conforto visual ajustável com apenas um clique.
* 🔊 **Leitura em Voz Alta:** Sistema inteligente que lê os blocos de texto selecionados por voz sintética.
* 🔤 **Controle de Tipografia (A+ / A-):** Botões dinâmicos que aumentam ou diminuem o tamanho da fonte com um **indicador de porcentagem em tempo real**.
* 📱 **Layout Totalmente Responsivo:** Design adaptável via *Media Queries* que funciona perfeitamente em computadores, tablets e celulares.
* 🛑 **Contraste e Legibilidade:** Paleta de cores selecionada criteriosamente para garantir uma leitura nítida e sem barreiras visuais.

---

## 🛠️ Tecnologias Utilizadas
O projeto foi construído do zero, priorizando uma codificação limpa, semântica e profissional (padrão de mercado):
* **HTML5:** Estruturação semântica avançada (`<header>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** Estilização moderna utilizando variáveis globais (`:root`), Flexbox para alinhamentos perfeitos e responsividade ativa.
* **JavaScript (Vanilla):** Lógica pura para manipulação do DOM, eventos de acessibilidade, cálculos de fontes e integração com a API de síntese de voz do navegador.

---

📚 Fontes de Pesquisa Científica e Dados Reais

Os argumentos e dados exibidos no site foram extraídos de bases de dados governamentais e instituições de altíssima credibilidade:
* Portal Oficial do Governo Brasileiro (**GOV.BR**)
* Empresa Brasileira de Pesquisa Agropecuária (**EMBRAPA**)
* Secretaria da Agricultura e do Abastecimento do Paraná (**SEAB-PR**)
* Instituto Brasileiro de Geografia e Estatística (**IBGE**)
